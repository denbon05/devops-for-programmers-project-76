## Usage

```bash
make touch-vault-secret-file
# then put vault secret key in .vault-secret

# install roles and collections
make galaxy-install

# setup requirements on targets
make setup

# deploy the app
make deploy
```

<p><b>Remember to encrypt vault files</b></p>
